# Orienta AI

Asistente de orientación educativa potenciado por inteligencia artificial. Proyecto JavaScript/Node.js.

## Stack

- **Lenguaje:** JavaScript
- **Runtime:** Node.js
- **Uso:** Orientación educativa con IA

## Quick Start

```bash
# Instalar dependencias
npm install

# Ejecutar
npm start

# Modo desarrollo
npm run dev
```
